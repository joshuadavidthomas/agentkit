# diataxis

An agent skill for writing and organizing documentation using the [Diátaxis](https://diataxis.fr/) framework.

The `SKILL.md` provides the core framework. Detailed guidance for each type lives in `references/`, downloaded from [diataxis.fr](https://diataxis.fr/) and converted to markdown for use as agent context.
