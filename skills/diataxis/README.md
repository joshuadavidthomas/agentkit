# diataxis

An agent skill for writing and organizing documentation using the [Diátaxis](https://diataxis.fr/) framework.

## What it does

When activated, this skill guides documentation authoring using the four Diátaxis documentation types:

- **Tutorials** — learning-oriented lessons
- **How-to guides** — goal-oriented directions
- **Reference** — information-oriented technical description
- **Explanation** — understanding-oriented discussion

The `SKILL.md` provides the core framework. Detailed guidance for each type lives in `references/` and is loaded on demand.

## References

The `references/` directory contains all pages from [diataxis.fr](https://diataxis.fr/), downloaded and converted to markdown for use as agent context.
